# 扫雷游戏 - 完整设计文档

## 项目概述
- **游戏类型**：经典扫雷（Minesweeper）
- **平台**：网页版（单文件 HTML/CSS/JavaScript）
- **视觉风格**：现代扁平化风格
- **目标**：打开即玩，零依赖

## 架构设计

### 文件结构
```
minesweeper/
└── index.html    # 单文件，包含 HTML + CSS + JavaScript
```

### 核心模块

#### 1. 游戏状态管理（Game State）
- `board`: 二维数组，存储每个格子的状态
  - `isMine`: 是否是雷
  - `isRevealed`: 是否已揭开
  - `isFlagged`: 是否标记旗帜
  - `adjacentMines`: 周围雷数
- `gameStatus`: 'idle' | 'playing' | 'won' | 'lost'
- `timer`: 游戏计时（秒）
- `flagsPlaced`: 已放置旗帜数
- `firstClick`: 是否首次点击

#### 2. 游戏逻辑（Game Logic）
- `initGame(rows, cols, mines)`: 初始化游戏
- `placeMines(excludeRow, excludeCol)`: 放置雷（排除首次点击位置）
- `calculateAdjacentMines()`: 计算每个格子周围雷数
- `revealCell(row, col)`: 揭开格子（递归展开空白区域）
- `toggleFlag(row, col)`: 切换旗帜标记
- `checkWin()`: 检查是否胜利
- `gameOver()`: 游戏结束处理

#### 3. UI 渲染（UI Renderer）
- `renderBoard()`: 渲染游戏面板
- `renderCell(cell, row, col)`: 渲染单个格子
- `updateTimer()`: 更新计时器显示
- `updateMineCounter()`: 更新剩余雷数显示
- `showGameOver(won)`: 显示游戏结束弹窗

#### 4. 事件处理（Event Handler）
- 左键点击：揭开格子
- 右键点击：标记/取消旗帜
- 难度按钮：切换难度
- 重新开始按钮

## 视觉设计（现代扁平化）

- **背景**：#f0f2f5（浅灰）
- **未揭开格子**：#4a90d9（蓝色），hover: #357abd
- **已揭开格子**：#ffffff（白色）
- **雷**：#e74c3c（红色）
- **旗帜**：#f39c12（橙色）
- **数字颜色**：
  - 1: #3498db（蓝）
  - 2: #2ecc71（绿）
  - 3: #e74c3c（红）
  - 4: #9b59b6（紫）
  - 5+: #34495e（深灰）

## 技术实现要点

### 1. 首次点击安全
- 第一次点击后才放置雷
- 确保首次点击位置及其周围 8 格无雷

### 2. 递归展开
- 点击空白格（周围 0 雷）时，自动展开周围 8 格
- 使用 BFS/DFS 避免栈溢出

### 3. 胜利条件
- 所有非雷格子都被揭开

### 4. 游戏结束
- 踩雷：显示所有雷，标记错误旗帜
- 胜利：显示庆祝动画
